About

This game was made as a project for a class in my college. It is written in C++ and uses Allegro 4 library. It's an arcade where you will have to take control of a ship and destroy all the enemy ships before they destroy you. The game is not complete because I had to turned it in, however it is playable and you will probably like it. A lot of enemy ships will come and attack you so you will have to come with an strategy in order to reach the higher levels.
